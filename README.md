# weekly-work-skill

Codex 周报撰写 skill：按日期从当前项目的 Codex 历史对话中提取工作内容，保存为结构化记录，并在需要时生成 Word 周报。

## 功能

- 按日期或日期范围提取项目工作内容
- 汇总指定日期所在周的周报
- 生成 `.docx` Word 文档
- 支持查看给其他 AI 使用的周报素材整理 prompt
- 支持更正、删除已保存记录
- 支持 macOS、Linux、Windows

## 安装

在 Codex 中安装 GitHub skill：

```text
安装 https://github.com/xjchen1/weekly-work-skill
